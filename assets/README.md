# Assets

This directory contains branding and visual assets for the RFC repository.

## Logo

The Claude World logo represents our community's focus on Claude Code excellence.

## Social Media Banner

Use the banner for sharing RFCs on social media.

## Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Primary Blue | `#339af0` | Links, highlights |
| Success Green | `#51cf66` | Positive outcomes |
| Warning Yellow | `#ffd43b` | Cautions, notes |
| Error Red | `#ff6b6b` | Problems, issues |
| Dark Gray | `#212529` | Text |
| Light Gray | `#f8f9fa` | Backgrounds |

## Usage Guidelines

- Always link back to [claude-world.com](https://claude-world.com)
- Include GitHub repo link when sharing
- Credit the original RFC author

## File Formats

- **SVG**: Preferred for logos (scalable)
- **PNG**: For social media (1200x630 for Open Graph)
- **WebP**: For web use (smaller file size)
